# Auth example

## How to use

### Download manually

```sh
$ curl https://codeload.github.com/bukinoshita/react-cookies/tar.gz/master | tar -xz --strip=2 master-canary/examples/with-auth
cd with-auth
```

Install and run:

```sh
$ yarn && yarn dev
```

### Run on CodeSandBox

Run the [`CodeSandBox` example](https://codesandbox.io/s/reactcookiesexample-y4fwl)
